# 中国宏观经济观察 | China Macro Observatory

**数据来源：[同花顺 iFinD](https://www.51ifind.com/) · 自动采集 · 每日更新**

一个自动更新的中国宏观经济时间线网站，包含：

- **📊 核心指标仪表盘** — GDP/CPI/PMI/社融/M2/LPR 等实时追踪
- **📈 趋势图表** — PMI+CPI走势、社融+M2对比、GDP季度增速
- **📰 政策要闻时间线** — 货币政策/财政政策/产业政策/房地产/国际贸易分类追踪

## 更新频率

- 每日 08:00 / 16:00 / 22:00 自动采集更新
